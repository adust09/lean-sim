# lean-sim

A collection of **interactive, in-browser simulators** that visualize the mechanisms
described in the chapters of [Lean Consensus](https://github.com/leanEthereum/leanSpec),
letting you run them in the browser to build intuition.
Implemented in vanilla JavaScript + HTML Canvas — no build step and no dependencies.

## Running

```bash
python3 -m http.server 8000
# open http://localhost:8000 in your browser
```
