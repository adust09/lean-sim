lean-sim

lean_consensus の各章で説明される仕組みを、ブラウザ上で動かして 直感的に理解するためのインタラクティブ・シミュレータ集です。

§2 · Simple Serialize
SSZ エンコーディング
シリアライズのオフセット、Merkleization とハッシュツリールート、 generalized index による証明の可視化。
§3 · The Time Model
時間モデル
スロットと4つのインターバル、リアルタイムのスロットクロック、 提案タイミングと締切のタイミングゲームを可視化。
§4 · State Transition
状態遷移関数
4 フェーズの遷移パイプライン (slot処理 → ヘッダ検証 → ペイロード実行 → state root検証) と、状態・ブロックの解剖図。
§5 · The Peer-to-Peer Layer
P2P レイヤー
Discovery v5 (Kademlia/XOR探索)、QUIC (HOLブロッキング/RTT)、 Gossipsub (mesh伝播)、Request-Response、そして新規ノードの 参加〜稼働ライフサイクルを横断する総まとめ。
§6 · Consensus Mechanisms
コンセンサス
source–target–head の関係、attestation のライフサイクルと集約、 justification / finalization の進行。Proposer の set-cover パッキングも。
§2–6 · 統括 (全章)
ビーコンチェーン稼働 / フォーク
全章を1本のスロット・ハートビートで統合した「生きたチェーン」。 さらに GHOST フォーク選択で、一時的フォーク・ネットワーク分断・ 二重提案など chain がフォークする複雑シナリオと reorg も再現。
